# MedCoPilot UI (демо)

Простой Streamlit-клиент поверх REST API: список диалогов, создание из текста,
переход в диалог и генерация SOAP-репорта по кнопке. Только для демо заказчику.

## Запуск

1. Поднять бэкенд (из корня репозитория):

   ```bash
   uv run uvicorn app.main:app --app-dir src --reload
   ```

2. Поднять UI (в отдельном терминале):

   ```bash
   uv run --project ui streamlit run ui/app.py
   ```

Адрес API настраивается в сайдбаре или через `MEDCOPILOT_API` (по умолчанию
`http://localhost:8000`). На старте бэкенд засевает демо-диалог, так что список
не пустой сразу.
